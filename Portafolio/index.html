PORTAFOLIO

Bienvenido a mi portafolio, soy Nicolás Cejas y acá verás mi relación con la programación.

He desarrollado (todos con JavaScript, HTML y CSS):

  • Proyecto 01 — Interactividad con botón

    Este proyecto incluye un botón interactivo que cambia visualmente al tocarlo (color/estado). El código HTML define el botón, CSS maneja el estilo y transiciones, y JavaScript añade el manejador de evento 'click' para alternar clases o estilos, produciendo el efecto visual al usuario.

  • Proyecto 02 — Galería de imágenes

    Proyecto que muestra una galería de imágenes (ej. fotos de gatos) con controles "siguiente" y "anterior" y una opción aleatoria. El HTML contiene la estructura de la imagen y botones, CSS define la apariencia y JavaScript gestiona el índice de la imagen actual, actualizando el elemento img y aplicando animaciones al cambiar.

  • Proyecto 03 — Formulario simple

    Contiene un formulario con campos como Nombre, Apellido y Edad y un botón "Enviar". El HTML define los inputs, CSS estiliza el formulario y JavaScript valida/gestiona el envío, por ejemplo mostrando los datos ingresados o previniendo el envío por defecto para procesarlos con DOM.

  • Proyecto 04 — Visualización de cartas/recursos gráficos

    Este proyecto muestra recursos gráficos (por ejemplo imágenes de cartas). El HTML carga las imágenes desde una carpeta `/lascartas/`, CSS controla la presentación y JavaScript puede añadir comportamiento (barajar, voltear o animar) según la interacción del usuario.

  • Página Jornada — Introducción a la programación

    Una página educativa que explica conceptos básicos de programación (bucles, condicionales, funciones, arrays) y contiene ejercicios interactivos con botones "Ejecutar" para ver resultados. HTML estructura la lección, CSS da estilo y JavaScript añade ejecución de ejemplos y comprobación de respuestas en las actividades.

Mi experiencia: tuve varias complicaciones al principio porque no tenía relación previa con la programación, pero gracias a mis profesores pude aprender y adaptarme rápidamente.

Me gustaria seguir apreniendo y mejorando mis habilidades en programación, explorando nuevas tecnologías y enfrentando desafíos más complejos en el futuro.
